# 介绍

MinecraftvsZombies2_City（MVZ2_City）是[https://space.bilibili.com/3546884492757767 痊愈加班费]以MC大战僵尸2为蓝本并从零开始制作的 MVZ2 同人游戏

本版本分为两种引擎制作的版本

使用GODOT Engine引擎制作的重制版和使用创游引擎 4.x制作的未重制版

如需游玩可在[https://github.com/jiabanfei1451/minecraftVSzombies2-CityGodot GitHub]下载预发布版源码游玩，目前仅有beta0.0.5标签拥有源代码压缩包
